# What's Changed 
- The user interface has been completely redesigned
- A new viewer component for examining BLOB cell values has been added
- All records recovered by FreeList are assigned to their original tables and displayed there (on a best-guess basis)
- Added support for a dark theme
- Database loading times have been drastically reduced
- Memory leaks have been fixed
- Smaller bug fixes in export functionality 
- Font-Dialog has been skipped
